# Microwave Oven Simulation using PICSimLab and MPLAB X IDE

## Overview

This repository contains the source code, project files, and documentation for a **Microwave Oven Simulation** developed using the **PIC16F877A** microcontroller. The project was designed and implemented in **MPLAB X IDE** using the **XC8 Compiler** and simulated in **PICSimLab**.

The simulation demonstrates the working principles of a microwave oven, including user input handling, cooking time management, countdown operation, and buzzer indication. The project showcases the use of embedded C programming, timers, interrupts, keypad interfacing, and LCD display control on the PIC16F877A microcontroller.

## Tools and Technologies

* **Microcontroller:** PIC16F877A
* **IDE:** MPLAB X IDE
* **Compiler:** XC8 Compiler
* **Simulation Software:** PICSimLab
* **Programming Language:** Embedded C

## Features

* User-friendly keypad interface
* LCD display for menu and timer
* Cooking time selection
* Countdown timer
* Start/Stop functionality
* Buzzer indication on completion
* Microwave door open/close simulation
* Real-time simulation using PICSimLab

## How to Run

1. Install **MPLAB X IDE** and the **XC8 Compiler**.
2. Install **PICSimLab**.
3. Open the project in MPLAB X IDE.
4. Build the project to generate the HEX file.
5. Load the HEX file into PICSimLab.
6. Run the simulation and test the microwave oven functionality.
